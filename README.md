# 📱 P4 - Pokédex (React Native)

Projeto final da disciplina de React Native da Residência TIC/Software do Serratec.

Aplicação mobile desenvolvida em React Native com Expo, criando uma Pokédex para consulta e visualização de informações dos Pokémons.

## 🚀 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- React Navigation
- Axios
- Async Storage
- Pokémon API

## ✨ Funcionalidades

- Listagem de Pokémons
- Tela de detalhes dos Pokémons
- Navegação entre telas
- Consumo de API externa
- Context API para gerenciamento de estado
- Persistência de dados local
- Login e cadastro do usuário
- Cadastrar novo Pokémon
- Pokedéx que armazena pokémons capturados
- Página de contato com informações dos criadores do projeto

## 📲 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/albino57/P4-Pokedex_ReactNative.git
````

Instale as dependências:
```bash
npm install
````

Execute o projeto:

````
npx expo start
ou
npx expo start --tunnel
````

O aplicativo pode ser executado utilizando o Expo Go ou um emulador mobile.
